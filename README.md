# 今天待辦事項管理系統

一個功能完整的待辦事項管理應用程序，支援 24 小時時間軸視圖、番茄鐘計時器、休息時間追蹤等功能。

## 功能特色

### 📅 月曆與日期管理
- 互動式月曆顯示，可切換月份
- 標記有待辦事項的日期
- 點擊日期可查看該日期的待辦事項
- 支援預先輸入未來日期的待辦事項

### ⏰ 24 小時時間軸視圖
- 直觀的 24 小時時間軸顯示
- 待辦事項直接對應到正確的時間位置
- 支援跨小時的待辦事項顯示
- 同一時段多個待辦事項自動並列顯示
- 按起始時間自動排序（由左而右）
- 待辦事項框可手動調整寬度
- 跨小時待辦事項使用不同顏色的左邊框標示

### ✏️ 待辦事項管理
- **新增待辦事項**：
  - 點擊時間段即可新增
  - 預設為整點開始，持續一小時
  - 可手動修改開始和結束時間
  - 支援跨小時的待辦事項
- **編輯待辦事項**：點擊待辦事項框即可編輯
- **刪除待辦事項**：在編輯模式下可刪除
- **完成待辦事項**：完成時顯示彩帶動畫效果
- **補充說明**：可為待辦事項添加備註

### 📊 統計功能
- **今天待辦事項統計**：
  - 共有幾項
  - 已完成幾項
  - 還剩下幾項
- **時間統計**：
  - 工作時間總計
  - 休息時間總計
- 統計區域可手動調整左右分隔線

### 🍅 番茄鐘計時器
- 可自訂工作時間長度（最長 1 小時）
- 指針式時鐘顯示
- 時間到時顯示全螢幕提醒：「站起來動一動，該喝水了」
- 提醒畫面包含動畫效果

### 💤 休息時間追蹤
- 可自訂休息時間（小時和分鐘，最長 4 小時）
- 時間到時顯示全螢幕提醒：「該回去工作了」
- 提醒畫面包含動畫效果

### 💭 心情與提醒
- 可記錄工作心情和提醒事項
- 文字框可隨內容自動調整大小
- 自動保存到本地儲存

### 🎨 介面特色
- 三欄式布局（月曆/統計、時間軸、計時器）
- 所有分隔線可手動調整大小
- 響應式設計
- 繁體中文介面

## 使用說明

### 新增待辦事項
1. 在 24 小時時間軸中，點擊想要新增待辦事項的時間段
2. 系統會自動填入整點開始時間和結束時間（預設一小時）
3. 如需修改時間，可在對話框中調整
4. 輸入事項內容
5. 點擊「儲存」

### 編輯待辦事項
1. 點擊時間軸中的待辦事項框
2. 在對話框中修改內容或時間
3. 點擊「儲存」

### 調整待辦事項寬度
- 將滑鼠移到待辦事項框的右邊緣
- 拖動即可調整寬度
- 調整後的寬度會自動保存
- 同一行的其他待辦事項會自動調整位置

### 完成待辦事項
1. 點擊待辦事項框
2. 點擊「完成」按鈕
3. 系統會顯示彩帶動畫效果
4. 待辦事項會被標記為已完成並從列表中移除

### 使用番茄鐘
1. 在右側面板設定工作時間（分鐘）
2. 點擊「開始」按鈕
3. 時間到時會顯示提醒畫面
4. 可隨時暫停或重置

### 使用休息時間
1. 在右側面板設定休息時間（小時和分鐘）
2. 點擊「開始」按鈕
3. 時間到時會顯示提醒畫面
4. 可隨時停止或重置

- **前端技術**：HTML5, CSS3, JavaScript (Vanilla JS)
- **資料儲存**：LocalStorage
- **瀏覽器支援**：現代瀏覽器（Chrome, Firefox, Edge, Safari）

## 資料儲存

所有資料（待辦事項、心情提醒等）都儲存在瀏覽器的 LocalStorage 中，不會上傳到伺服器。

## 注意事項

- 資料僅儲存在本地瀏覽器中，清除瀏覽器資料會導致資料遺失
- 建議定期備份重要資料
- 跨小時的待辦事項在所有涉及的小時內會顯示相同寬度
- 待辦事項按起始時間自動排序
